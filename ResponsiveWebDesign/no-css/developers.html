Open Developer Positions

All candidates should be naturally motivated individuals who seek to constantly learn and improve their technical skills. The individuals that we seek have a passion for technology and motivated to seek better solutions to business problems through technology.

Java Developer


Highly motivated, technically driven individual with 1-5 years of development with the Java platform (version 1.5 and above). This individual must have sound programming background with a firm grasp of object oriented concepts and techniques. An understanding of industry standard design patterns and best practices with the Java platform is desirable.


Requirements:

  • Experience developing applications on the Java 5 platform or later
  • Knowledge of object oriented concepts, patterns, and practices
  • Experience with a RDBMS (Oracle, SQL Server, MySQL) experience writing SQL

Desirable:

  • Experience using a Java web application framework (Struts, Wicket, Spring MVC, Tapestry, etc)
  • Experience with an application build framework (Ant, Maven, Gradle)
  • Experience using a JEE server (Tomcat, WebLogic, WebSphere, Glassfish, JBoss, etc)
  • Experience building automated unit tests (JUnit or TestNG frameworks) in a continuous integration development project

.NET Developer


Highly motivated, technically driven individual with 1-5 years of development with the .NET platform (3.0 and above). This individual must have sound programming background with a firm grasp of object oriented concepts and techniques. An understanding of industry standard design patterns and best practices with the .NET platform is desirable.


Requirements:

  • Experience developing applications with .NET 3.0 framework or later
  • Knowledge of object oriented concepts, patterns, and practices
  • Experience with a RDBMS (Oracle, SQL Server, MySQL) experience writing SQL

Desirable:

  • Experience using a .NET web application framework (ASP MVC)
  • Experience with an application build framework (MSBuild, NAnt)
  • Experience building automated unit tests (NUnit or MSTest frameworks) in a continuous integration development project
